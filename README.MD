### Construire l'image et lancer un conteneur sur l'image
```console
 docker compose up
 ```

 ```console
 docker compose build
 ```

### 
```console
docker compose run 'nom du service'
```

### Couper et supprimer les container

```console 
docker compose down
```



### Kill apache sur l'host

```console 
sudo apachectl stop
```
